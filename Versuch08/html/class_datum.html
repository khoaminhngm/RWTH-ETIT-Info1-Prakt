Detailed Description

Klasse für ein Datum.

Speichert ein Datum, bestehend aus Tag, Monat und Jahr. Die Klasse Datum bietet verschiedene Funktionalitäten/Operatoren die zum Ausgeben/Bearbeiten/Vergleichen von verschiedenen Daten genutzt werden können

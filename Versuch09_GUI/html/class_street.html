Detailed Description

Repräsentiert eine Verbindung (Straße) zwischen zwei Städten.

Die Klasse verwaltet zwei Zeiger auf City-Objekte, zwischen denen sie eine Linie in einer QGraphicsScene zeichnen kann. Optional kann die Straße auch in Rot hervorgehoben werden.

Detailed Description

Dialogfenster zum Erstellen und Hinzufügen einer Straße zwischen zwei Städten.

Diese Klasse bietet eine Benutzeroberfläche zur Eingabe von Start- und Zielstadt. Nach Bestätigung wird eine Street zwischen den angegebenen Städten erstellt, sofern beide in der Map vorhanden sind.

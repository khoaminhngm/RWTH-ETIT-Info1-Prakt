Detailed Description

Dialogfenster zum Erstellen und Hinzufügen einer neuen Stadt zur Karte.

Diese Klasse bietet eine einfache Eingabemaske zur Erfassung des Stadtnamens sowie der X- und Y-Koordinaten. Nach Bestätigung wird ein neues City-Objekt erstellt.

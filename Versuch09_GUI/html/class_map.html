Detailed Description

Repräsentiert eine Karte, bestehend aus Städten und Straßen.

Die Klasse verwaltet eine Liste von City-Objekten und Street-Objekten, stellt Methoden zur Visualisierung auf einer Szene bereit und unterstützt Pfadsuche, Verbindungsmanagement und Distanzberechnungen.

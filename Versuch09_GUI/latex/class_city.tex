\doxysubsection{Detailed Description}
Repräsentiert eine Stadt mit Namen und Koordinaten im 2D-\/\+Raum. 

Die Klasse stellt Methoden bereit zur Visualisierung auf einer QGraphics\+Scene sowie Zugriff auf Positions-\/ und Namensinformationen.
